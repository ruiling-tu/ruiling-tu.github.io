The blog is built around Markdown, so each essay can stay close to plain text. Images
live in `public/images/` and can be inserted anywhere inside the writing.

## A section heading

The layout keeps the text column narrow, lets images interrupt the rhythm, and leaves
wide margins around the page. That gives longer essays a little more silence.
